# Yaguniv Expo Frontend

Baseball match schedule and community app built with Expo and React Native.

## Tech Stack
- Framework: Expo (Latest Stable)
- Language: TypeScript (Strict Mode)
- Styling: NativeWind (TailwindCSS for RN) / StyleSheet
- State Management: Zustand, TanStack Query (v5)
- Navigation: Expo Router
- Storage: expo-secure-store, mmkv

## Project Rules
- Strict Typing: No `any` allowed.
- Functional Components only.
- Separate business logic into Custom Hooks.
- No inline styles (use StyleSheet or NativeWind).
- Explicit data fetching (prevent N+1).
